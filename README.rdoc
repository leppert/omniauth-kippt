= OmniAuth::Kippt

OmniAuth stratgies to pull user profile from Kippt through XSI api.

== Usage

Install manually or using Bundler:

    gem 'omniauth-kippt'

Add :kippt provider to omniauth builder:

    # configuration for xdp sandbox
    use OmniAuth::Builder do
      provider :kippt, "http://xsp1.xdp.broadsoft.com/com.broadsoft.xsi-actions/v2.0", domain: 'xdp.broadsoft.com'
      # provider ...
    end
